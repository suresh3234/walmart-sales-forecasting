# 📊 Walmart Sales Forecasting

## 🚀 Project Overview

This project analyzes Walmart retail sales data to identify trends, understand key business drivers, and forecast future sales using machine learning.

---

## 🎯 Objectives

* Perform Exploratory Data Analysis (EDA)
* Identify sales patterns and seasonality
* Analyze impact of holidays and external factors
* Build forecasting models for weekly sales
* Create interactive dashboard for business insights

---

## 📁 Dataset

* ~45,000+ records
* 20+ features including:

  * Store, Dept, Weekly Sales
  * Temperature, Fuel Price
  * CPI, Unemployment
  * Holiday indicators

---

## 🧹 Data Processing

* Handled missing values and outliers
* Converted data types and created date features
* Cleaned and prepared dataset for analysis

---

## 📊 Exploratory Data Analysis

* Sales trends over time
* Store and department performance
* Holiday vs non-holiday impact
* Correlation analysis

---

## 🤖 Machine Learning Models

* Linear Regression
* Random Forest

### 📈 Evaluation

* Mean Absolute Error (MAE)

---

## 📊 Dashboard

Built using Power BI:

* KPI Cards (Total Sales, Avg Sales)
* Sales Trend Analysis
* Store Performance
* Forecast Visualization

---

## 📈 Key Insights

* Sales peak during holidays and weekends
* Certain stores consistently outperform
* External factors influence sales patterns
* Seasonal trends observed

---
